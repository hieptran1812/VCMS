# Installation

## Chạy docker images

Link docker image: https://hub.docker.com/repository/docker/hieptran1812/vcms/general

Chạy lệnh sau đây để pull:

```
docker pull hieptran1812/vcms:latest
```

Chạy lệnh dưới để run image:

```
docker run --gpus all -d -it -p <port>:<port> -v <đường dẫn tới thư mục code>:/app vcms
```

Execute docker container:

```
docker exec -it <container id> /bin/bash
```

## Download weights

Tải các file weight tại đây: [link](https://drive.google.com/drive/folders/16N6hIT5XAI2tb0i9ntZ1M6BGGxBtDgMU?usp=sharing). Giải nén và chuyển các file weight vào thư mục checkpoints.

## Chạy chương trình

Sau khi execute container, ta chạy lệnh sau:

```
cd src
python3 serve.py
```
